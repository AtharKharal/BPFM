Reading Data

All the values marked as ‘?’ have been read-in as NA’s using built-in feature of pandas read_csv method.

import pandas as pd
dt = pd.read_csv('Blood_Pressure_data.csv', na_values='?' )
## sys:1: DtypeWarning: Columns (10) have mixed types.Specify dtype option on import or set low_memory=False.
